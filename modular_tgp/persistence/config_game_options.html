/tg/ Station 13 - Modules - Types

modular_tgp/persistence/config_game_options.dm

/datum/config_entry/flag/persistent_save_enabledThis will load the most recent saved maps, z-levels, and jsons in the _maps/persistence folder organized by year/month/day/hour-minute-second
/datum/config_entry/keyed_list/persistent_save_z_levelsInclude specific z-levels when saving and loading determined by trait
/datum/config_entry/keyed_list/persistent_save_flagsInclude specific save flags that determines types and data to save and load
/datum/config_entry/flag/persistent_use_static_map_gridIf enabled, disables procedural grid generation and loads the pre-configured layout from z-level JSONs.
/datum/config_entry/number/persistent_autosave_periodPeriod of time in hours between map autosaves (set to -1 to only allow saving when server reboots)
/datum/config_entry/number/persistent_max_object_limit_per_turfThe max amount of objects that can be saved on a single turf (rest get skipped)
/datum/config_entry/number/persistent_max_mob_limit_per_turfThe max amount of mobs that can be saved on a single turf (rest get skipped)
/datum/config_entry/number/persistent_max_autosavesThe maximum amount of autosaves to store (set to -1 to never delete any autosaves)
